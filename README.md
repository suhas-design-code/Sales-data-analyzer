# Sales Data Analyzer

##  Overview
This project analyzes sales data to find insights like top-performing products, revenue trends, and customer patterns.

## Features
- Data cleaning using Python (pandas)
- Data visualization (Matplotlib/Seaborn)
- Automated reports
- Optional Power BI dashboard

##  Tech Stack
Python, pandas, matplotlib, seaborn

##  How to Run
1. Clone the repo  
2. Run `main.py`  
3. Check the generated output
